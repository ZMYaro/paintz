Keyboard shortcuts

Document actions
Ctrl+N Clear drawing (standalone only)
Ctrl+Shift+N Clear drawing immediately (standalone only)
Ctrl+O Open image
Ctrl+S Save as
Ctrl+P Print
Ctrl+Z Undo
Ctrl+Y Redo
Ctrl+E Crop or scale canvas
Ctrl+V Paste
Ctrl+Alt+V Paste from file
Ctrl+Alt+= Zoom in
Ctrl+Alt+- Zoom out
Ctrl+Alt+0 Zoom 100%
 
Tools
P Pencil
B Brush
L Line
C Curve
R Rectangle
O Oval
H Pan (hand)
S Selection
F Freeform selection
E Eraser
K Flood fill (paint bucket)
I Eyedropper (color picker)
T Text
 
Drawing tool actions
[ Decrease line width
] Increase line width
X Switch line and fill colors
 
Selection tool actions
Ctrl+A Select all
Esc Cancel selection/text box
Delete or Backspace Delete selection
Ctrl+D Duplicate selection
Ctrl+X Cut selection
Ctrl+C Copy selection
Ctrl+I Invert colors
 
Text tool actions
Ctrl+B Bold text
Ctrl+I Italicize text
Ctrl+U Underline text
Alt+Shift+5 Strike text
Ctrl+Enter Finalize text
 
App actions
Ctrl+F11 View drawing in full screen (standalone only)
Ctrl+G Show/hide gridlines
F1 Open help
? (Shift+/) Open keyboard shortcut reference
Ctrl+Alt+Shift+? Open MS Paint access key reference
