MS Paint access keys

MS Paint access keys are only available when PaintZ is running in its own window as a standalone app.

When using MS Paint access keys, continue pressing the keys until the sequence is completed. Pressing Esc or a key that is not part of any sequence will cancel the sequence.

If an access key sequence did something different in Windows 7 Paint than classic Paint, PaintZ will match classic Paint.
